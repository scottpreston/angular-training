# Cordova # 

Quick Start

`npm install -g cordova`

Create A Project

```bash
cordova create testCordova
cd testCordova
cordova platform
cordova install
```

Add Platform iOS

```bash
cordova platform add ios
cordova emulate ios
```

Add Platform Browser

```bash
cordova platform add browser
cordova run browser
```
